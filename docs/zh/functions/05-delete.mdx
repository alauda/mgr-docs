# 删除实例

::: tip 删除的资源
删除实例会自动删除由实例创建的可以安全删除的资源, 例如 Statefulset, Deployment, ConfigMap, Secret, Service。
可能包含重要数据的特殊资源需要手动删除, 例如 PersistentVolumeClaim, MySQL-MGR 的备份资源 mysqlbackups.mysql.middleware.alauda.io 与其关联的实际数据。
:::

## 操作步骤
<Tabs>
<Tab label="CLI">
1. 删除实例
```bash
kubectl delete mysql -n ${namespace} ${name}
```

2. 清理持久化数据
```bash
kubectl get pvc -n ${namespace} -l v1alpha1.mysql.middleware.alauda.io/cluster=${name} -o name | xargs kubectl delete -n ${namespace}
```
</Tab>

<Tab label="Web Console">
1. 进入实例 **概览**
2. 点击 **删除实例**
3. 选择删除选项：
   - 可选项：删除持久卷声明
4. 确认删除：
   - 关联资源检查
   - 输入实例名称最终确认删除
</Tab>
</Tabs>

# 日志

## 功能介绍
从容器层面了解实例运行过程中的日志，例如 MySQL 错误日志。妥善使用日志能帮助您快速定位问题，处理故障和异常。

## 操作步骤
<Tabs>
<Tab label="CLI">
1. 查看实例所有相关 pod，包含 mysql pod 和 mysql-router pod。
```bash
kubectl get pod -n ${namespace} -l "middleware.instance/name=${instance_name}"
```
2. 查看某个 pod 日志查询
```bash
kubectl logs -n ${namespace} ${pod_name}
```
</Tab>

<Tab label="Web Console">

1. 在左侧导航栏中，单击 **MySQL-MGR**。

2. 单击 ***命名空间的名称***。

3. 单击 ***实例名称*** 。
    
4. 在 **实时日志** 页签中，设置查看选查看对应日志。
</Tab>
</Tabs>

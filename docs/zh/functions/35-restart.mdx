# 实例重启

## 操作步骤
<Tabs>
    <Tab label="CTL">
        1. 添加注解标记重启实例
        ```bash
        kubectl annotate mysql $name -n $namespace "restartTime=$(date +%s)"
        ```
    </Tab>

    <Tab label="Web Console">
        1. 在左侧导航栏中，单击 **MySQL-MGR**。

        2. 单击 ***命名空间的名称***。

        2. 单击 ***实例名称*** 。

        3. 单击 **操作** > **重启**，并确认。
    </Tab>
</Tabs>

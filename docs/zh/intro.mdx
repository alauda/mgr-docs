# 介绍

Alauda Database for MySQL-MGR 是基于MySQL Group Replication技术的高可用数据库解决方案，运行在Kubernetes上。它提供了自动化部署与管理MySQL Group Replication集群的能力。

## 核心优势

- **高可用性**：基于MySQL Group Replication提供多主复制和高可用性
- **自动化管理**：通过Operator自动化管理MySQL集群
- **弹性扩展**：支持在线扩展和缩容
- **数据一致性**：基于Paxos协议保证数据强一致性
- **读写分离**：支持自动读写分离

## 应用场景

- 需要高可用性的关键业务数据库
- 需要多主写入的分布式数据库场景
- 需要弹性扩展的云原生数据库

## 使用限制

- 支持MySQL 8.0及以上版本
- 建议至少3个节点以确保高可用性
- 需要Kubernetes 1.19及以上版本

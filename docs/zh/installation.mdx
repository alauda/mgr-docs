# 安装

## 安装 Alauda Application Services Essentials

::: tip 可选安装
插件为 Alauda Application Services 界面化服务基础组件，无需界面化服务时可以不安装。
:::

### 前提条件
1. 下载与平台匹配的 Alauda Application Services Essentials 安装包。
2. 通过平台应用上架能力，上架 Alauda Application Services Essentials 至 `global` 集群。

### 操作步骤
<Tabs>
<Tab label="Web Console">
1. 在左侧导航栏中，进入 **应用商店管理** > **集群插件**。
2. 选择 `global` 集群
3. 单击 **Alauda Application Services Essentials** 插件右侧的操作按钮 > **安装**。
</Tab>
</Tabs>

## 安装 Alauda Application Services RDS Framework

::: tip 可选安装
插件为 Alauda Application Services 界面化服务基础组件，无需界面化服务时可以不安装。
:::

### 前提条件
1. 下载与平台匹配的 Alauda Application Services RDS Framework 安装包。
2. 通过平台应用上架能力，上架 Alauda Application Services RDS Framework 至 任意希望使用 Alauda Application Services 界面化能力的集群。

### 操作步骤
<Tabs>
<Tab label="Web Console">
1. 登录平台，进入 **平台管理** 页面。
2. 左侧导航栏选择 **应用商店管理** -> **Operator Hub**， 进入 **Operator Hub** 页面。
3. 找到 **Alauda Application Services RDS Framework**，点击 **安装**， 进入安装页面。

配置参数:

|**参数**| **推荐配置** |
|:--------|:------------|
| **频道**    | 默认 Channel 为 **stable** 。|
| **版本**    | 选择期望安装的版本。|
| **安装模式** | **集群**：集群下所有命名空间共用一个 Operator 对实例进行创建和管理，资源占用更低。|
| **安装位置** | 选择 **推荐**：无则自动创建。|
| **升级策略** | **手动**：Operator Hub 有新版本时，需要手动确认升级至 Operator 至最新版本。|

4. 在 **安装 Operator** 页面，选择 **默认配置**， 点击 **安装**，完成 **Alauda Application Services RDS Framework** 安装。

</Tab>
</Tabs>

## 安装 Alauda Database for MySQL-MGR 插件

### 前提条件
1. 下载与平台匹配的插件安装包。
2. 通过平台应用上架能力，上架插件至任意希望使用 Alauda Database for MySQL-MGR 组件能力的集群。

### 操作步骤

<Tabs>
<Tab label="Web Console">
    1. 登录平台，进入 **平台管理** 页面。
    2. 左侧导航栏选择 **应用商店管理** -> **Operator Hub**， 进入 **Operator Hub** 页面。
    3. 找到 **Alauda Database Service for  MySQL**，点击 **安装**， 进入安装页面。

    配置参数:

    |**参数**| **推荐配置** |
    |:--------|:------------|
    | **频道**    | 默认 Channel 为 **alpha** 。|
    | **版本**    | 选择期望安装的插件版本 。|
    | **安装模式** | **集群**：集群下所有命名空间共用一个 Operator 对实例进行创建和管理，资源占用更低。|
    | **安装位置** | 选择 **推荐**：无则自动创建。|
    | **升级策略** | **手动**：Operator Hub 有新版本时，需要手动确认升级至 Operator 至最新版本。|

    4. 在 **安装 Operator** 页面，使用默认配置， 点击 **安装**，完成安装。
</Tab>
</Tabs>

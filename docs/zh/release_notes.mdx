# 发行说明

## v4.0.0

### 新增和优化的功能

#### 组件与平台解耦

Alauda 数据库服务（MySQL-MGR）的生命周期已从平台中解耦，这意味着当客户需要某个功能或修复错误时，MySQL-MGR 团队可以立即发布，而无需等待平台的整体发布。

#### 组件详情页面的实时日志

MySQL-MGR 详情页面上的日志已被实时日志替换，这些日志会即时更新，便于故障排除和问题分析。

# Обновление

Служба базы данных Alauda для MySQL-MGR будет выполнять обновления в зависимости от настроенной стратегии обновления:

- **Автоматически** : Автоматические обновления запускаются немедленно при обнаружении новых версий компонентов.
- **Вручную** : Требуется ручное одобрение перед началом процесса обновления.

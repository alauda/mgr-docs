# Перезапуск экземпляра

## Процедура

<Tabs>
  <Tab label="CTL">
    1. Добавьте аннотацию, чтобы отметить экземпляр для перезапуска

    ```bash
    kubectl annotate mysql ${name} -n ${namespace} "restartTime=$(date +%s)"
    ```
  </Tab>

  <Tab label="Web Console">
    1. Нажмите на **MySQL-MGR** в левой боковой панели.

    2. Нажмите на ***имя пространства***.

    3. Нажмите на ***имя экземпляра***.

    4. Нажмите **Действия** > **Перезапуск**, и подтвердите.
  </Tab>
</Tabs>

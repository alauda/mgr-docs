# Анализ запросов

## Введение в функцию

Анализируя активность запросов SQL, точно определите SQL-операторы, которые влияют на производительность базы данных, или непрерывно отслеживайте активность запросов и выполняйте корректировки и оптимизацию операторов.

## Процедура

<Tabs>
  <Tab label="Web Console">
    1. В левом навигационном меню нажмите на **MySQL-MGR**.

    2. Нажмите на ***Название пространства имен***.

    3. Нажмите на ***Название экземпляра***.

    4. На вкладке **Анализ запросов** установите критерии поиска.

       - Чтобы найти операторы, выполненные в определенном диапазоне времени: выберите временной период в **Диапазон времени**.

       - Чтобы найти конкретные операторы: введите ключевое слово оператора в **Поиск по операторам SQL**, затем нажмите поиск.

    5. Подождите немного и проверьте результаты.
  </Tab>
</Tabs>

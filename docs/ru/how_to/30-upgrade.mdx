# Обновление патч-версии

С выпуском исправлений уязвимостей или новых функций могут происходить обновления патч-версии для промежуточного программного обеспечения. Во время обновлений патч-версий будут инициированы последовательные обновления. Рекомендуется выбрать подходящее временное окно для обновления. Когда патч-версия доступна для обновления, рекомендуется обновить её как можно скорее, чтобы применить последние исправления к компонентам.

## Настройка стратегии обновления патч-версии

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Перейдите на страницу деталей экземпляра.

    2. Нажмите на **Стратегия обновления патч-версии**.

    3. Выберите **Ручной** или **Автоматический**.

    4. Нажмите **Обновить**, чтобы завершить настройку.
  </Tab>
</Tabs>

## Обновление патч-версии

### Предварительные условия

Когда патч-версия доступна, на странице списка экземпляров будет отображаться иконка обновления, а на странице деталей экземпляра будет показано уведомление об обновлении.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Перейдите на страницу деталей экземпляра.

    2. Нажмите **Обновить сейчас** рядом с уведомлением об обновлении.

    3. Прочитайте уведомление об обновлении, чтобы понять изменения в версиях компонентов и плагинов.

    4. Нажмите **Обновить сейчас**.

    5. Дождитесь изменения статуса экземпляра на "работает", что указывает на успешное обновление.
  </Tab>
</Tabs>

## Массовое обновление патч-версии

### Предварительные условия

Пользователи должны иметь права администратора платформы.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. В левой панели навигации нажмите на **Управление обновлениями**.

    2. Выберите необходимого **оператора**, чтобы перейти на страницу деталей оператора.

    3. Выберите экземпляры, которые необходимо обновить, и нажмите **Массовое обновление**.

    4. Прочитайте уведомление об обновлении, чтобы понять изменения в версиях компонентов и плагинов.

    5. Нажмите **Обновить сейчас**.

    6. Дождитесь изменения статуса экземпляра на "работает", что указывает на успешное обновление.
  </Tab>
</Tabs>

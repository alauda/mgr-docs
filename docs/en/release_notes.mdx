## v4.3.1

### New and Optimized Features

- Removed deprecated MySQL-PXC operator integration. MySQL-MGR is now the sole supported architecture.
- Support for deploying MGR clusters on non-TopoLVM environments using local storage.
- Enhanced RBAC security with least-privilege permissions for MGR cluster components.
- Fixed S3 backup EntityTooLarge issue by switching to multipart upload for large files.

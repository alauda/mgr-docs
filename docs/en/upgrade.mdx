# Upgrade

Alauda Database for MySQL-MGR adheres to the platform upgrade process without requiring any special handling. For upgrading Alauda Application Services, refer to the <ExternalSiteLink name="acp" href="/upgrade/index" children="Upgrading the Operator" />. Once the upgrade has been completed, you may follow the [Minor Version Upgrade Guide](how_to/30-upgrade) to upgrade the MySQL-MGR instances.
